### 四、简化架构层级（从上到下）

1. 用户层：PC 管理后台 → AGUI 对话面板
2. 前端层：React + AntD Pro + A2UI 执行引擎
3. 接口层：网关 + 权限 + 审计日志
4. AI 服务层：意图识别 + RAG 链路 + FunctionCalling 链路
5. 数据层：业务库 + 数仓 + 向量库 + 缓存
6. 模型层：商用 / 私有大模型

### 五、落地优先级（从高到低）

1. 基础框架：前端（React+AntD Pro）+ 后端（主力语言框架）+ 核心存储（业务库 + Redis）
2. AI 核心：AGUI + 商用大模型 + RAG（优先实现知识问答）
3. 核心能力：FunctionCalling（数据查询）+ A2UI（智能操作）
4. 进阶能力：数仓搭建 + 经营诊断模块
5. 优化迭代：私有模型部署、自研编排引擎、个性化交互

### 六、关键约束与要求

- 安全性：操作权限隔离、查询审计、知识库访问控制
- 稳定性：大模型降级策略、服务熔断、数据备份
- 可扩展性：支持模型切换、存储扩容、能力插件化
