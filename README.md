# AIGovern Pro (智管通 AI) - 项目介绍

## 一、项目定位

AIGovern Pro 是一款 **AI 原生企业级 B 端管理系统**，聚焦企业经营管理全流程，通过「知识问答、数据查询、智能操作、经营诊断」四大核心能力，打造 “能听懂、能查询、能操作、能分析” 的智能管理中枢，助力企业降本增效、决策提效。

## 二、英文名称释义

- **核心名称**：AIGovern Pro
- 释义：
  - “AI”：突出人工智能原生技术特性（AGUI+A2UI+RAG+FunctionCalling）；
  - “Govern”：对应 “管理” 核心场景，覆盖企业经营、流程、数据的全维度管控；
  - “Pro”：代表专业级（Professional），强调企业级稳定性、安全性与扩展性。
- 简称：AG Pro（适配内部沟通、系统命名、域名注册）

## 三、核心价值

### 1. 效率提升

- 知识问答：3 秒检索企业内部文档、制度、FAQ，替代人工查询；
- 智能操作：AI 自动完成表单填写、流程触发、审批处理，减少重复操作；
- 数据查询：自然语言直接获取报表、指标，无需懂 SQL/API。

### 2. 决策赋能

- 经营诊断：自动分析营收、毛利、流失等核心指标，提供异常预警与归因建议；
- 全维数据贯通：整合业务数据、知识库、流程节点，打破信息孤岛。

### 3. 技术领先

- AI 原生交互：AGUI 对话面板 + A2UI 智能操作引擎，适配 B 端用户使用习惯；
- 稳定可靠：商用大模型 + RAG 检索增强，抗幻觉、可溯源、高可用。

## 四、核心功能模块


| 模块                       | 核心能力                                  | 应用场景举例                      |
| ------------------------ | ------------------------------------- | --------------------------- |
| 知识问答（Knowledge QA）       | 企业文档、制度、FAQ、产品手册智能检索与问答，支持多轮交互、溯源引用   | 新员工查询入职流程、客服查询产品规则、运维查询配置文档 |
| 数据查询（Data Query）         | 自然语言转 SQL/API，查询订单、用户、营收等业务数据，生成可视化报表 | 运营查月度活跃用户、财务查区域毛利、销售查客户跟进数据 |
| 智能操作（Smart Operation）    | AI 自动执行系统操作：新增数据、审批流程、导出文件、配置参数、触发任务  | 自动审批常规报销、批量导出客户名单、配置定时数据同步  |
| 经营诊断（Business Diagnosis） | 核心指标监控、异常识别、根因分析、优化建议，支撑经营决策          | 营收下滑归因、高流失客户预警、毛利提升策略推荐     |


## 五、核心技术架构

### 1. 技术栈核心

- 前端：React 18 + TypeScript + Ant Design Pro + AGUI（对话交互）+ A2UI（智能操作引擎）
- 后端：Java/Go/Python（企业主力语言）+ SpringBoot/Gin/FastAPI
- 数据层：MySQL/PostgreSQL（业务库）+ Redis（缓存）+ Milvus/Chroma（向量库）+ ClickHouse/Doris（数仓）
- AI 中台：商用大模型（豆包 / 通义千问 / 文心一言）+ LangChain（编排）+ RAG（检索增强）+ FunctionCalling（工具调用）

### 2. 架构优势

- 兼容性：适配企业现有技术栈，无需重构；
- 安全性：权限隔离、操作审计、数据加密，符合企业合规要求；
- 扩展性：支持模型切换、存储扩容、功能插件化迭代。

## 六、目标用户与应用场景

### 1. 目标用户

- 企业内部管理者：CEO、部门负责人（经营诊断、决策支持）；
- 业务运营人员：运营、销售、财务、客服（数据查询、知识问答）；
- 行政 / IT 人员：行政、运维、HR（智能操作、流程自动化）。

### 2. 典型应用场景

- 日常办公：快速查询知识、自动处理重复操作，提升工作效率；
- 经营分析：定期查看核心指标、识别业务异常，辅助决策；
- 新人上手：快速获取岗位相关知识与操作指引，降低培训成本；
- 跨部门协同：打破数据与知识壁垒，提升协同效率。

## 七、项目愿景

成为企业经营管理的「智能中枢」，让 AI 深度融入每一个管理环节，推动企业从 “人工操作” 向 “智能决策” 升级，实现降本增效、持续增长。

## 八、快速启动

### 前端启动

```bash
cd frontend
pnpm install
pnpm dev
```

访问 http://localhost:3001

### 后端启动

```bash
cd backend
pip install -r requirements.txt
cp .env.example .env

# 方式一：使用启动脚本
python run.py

# 方式二：直接使用 Uvicorn
python -m uvicorn app.main:app --reload --port 8000
```

访问 API 文档：http://localhost:8000/docs

详见 [后端快速启动指南](backend/QUICKSTART.md)
